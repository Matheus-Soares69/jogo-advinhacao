# Jogo da Adivinhação

Jogo feito em Python onde o computador sorteia um número entre 1 e 100 e o usuário tenta adivinhar.

## Como jogar

1. Tenha o Python instalado no seu computador
2. Abra o terminal na pasta do projeto
3. Execute o comando abaixo:

```bash
python Jogo_Adivinhacao.py
```

## O que aprendi

- Funções com def e return
- Loops com while
- Condicionais com if/elif/else
- Entrada do usuário com input()

## Tecnologias

- Python
- VScode
- Biblioteca random
