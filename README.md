# JOGO DE ADVINHAÇÃO

Projeto simples de um jogo de adivinhação feito em python, criado para melhorar e fixar meu aprendizado sobre, funções loops e condicionais

## Como jogar
```bash
python  Jogo_de_Adivinhação.py
```


## Oq eu aprendi

- Funções com def e return
- Loops com while
- Condicionais com if/ elif/ else
- Entrada do usuário com input()

## Funcionalidades( Adicionadas )

- Sistema de dificuldade
- Validação de entrada
- Menu interativo
- Controle de tentativas

## Tecnologias
